About This Project

This tool was created as a learning exercise to understand how threat intelligence can be automatically converted into structured formats. It demonstrates natural language processing, web development, and cybersecurity concepts.

Technologies Used: Python Flask, RapidFuzz, stix2, spaCy, JavaScript, HTML/CSS, and many others.

What is STIX?

Structured Threat Information Expression (STIX) is a standardised language for describing cyber threat information to enable threat sharing and automated analysis.

Why Use This Tool?

Eliminates manual STIX creation, reduces errors, and enables real-time threat intelligence sharing across security teams and platforms.

Supported Indicators

Detects IP addresses, domains, email addresses, file hashes, malware types, attack patterns, and threat actor information.
